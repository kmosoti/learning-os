# Architecture Decision Records

This directory contains durable Architecture Decision Records for decisions that are expensive to reverse or that shape multiple implementation cards.

The older [Decision Log](../05-decision-log.md) remains a lightweight historical summary. New significant decisions should get a full ADR here and a short summary in the decision log.
